This project tries to mix all that has been learned up to this point into a useful application: 
computing the square of a complex number (using floating point numbers for the I/O data). 

The most important part in this example is the casting of the input (from float to u32) and the 
output (from u32 to float) parameters to accomodate the data types defined in the C driver API generated by Vivado HLS. For a reference check ug902-vivado-high-level-synthesis.pdf (C Driver Files and Float types, pag. 119). 


Status: works correctly.
